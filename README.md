## README — Repositorio de Releases

Este repositorio es un **canal de distribución únicamente** para lanzamientos de productos. **No contiene código fuente.**

### Propósito

Proveer un registro centralizado y versionado de entregables (binarios, paquetes, artefactos) para consumo por clientes, pipelines de CI/CD o sistemas de despliegue.

### Contenido

* Releases etiquetados siguiendo versionado semántico (`vX.Y.Z`)
* Artefactos de build (ej. `.apk`, `.ipa`, `.zip`, `.tar.gz`)
* Notas de versión y changelogs
* Checksums/firmas cuando aplique

### Convenciones

* **Versionado:** Semantic Versioning (SemVer)
* **Tag = Release:** Cada tag corresponde a un release publicado
* **Inmutabilidad:** Los artefactos no se modifican después de publicados
* **Nomenclatura:** `<producto>-<version>-<plataforma>.<ext>`

### Uso

1. Ir a **Releases**
2. Seleccionar la versión deseada
3. Descargar el artefacto correspondiente

### Notas

* Para el código fuente, consultar el repositorio principal de desarrollo
* Issues y solicitudes de features deben reportarse en el repositorio de código fuente, no aquí
